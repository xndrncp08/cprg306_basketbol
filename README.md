Basketbol
NBA statistics web application inspired by NBA 2K's interface and bold visual design. Delivers real-time NBA data with a game inspired aesthetic.

Features
  Player Search & Profiles: Search NBA players with autocomplete, view detailed stat cards
  Team Standings: Live conference standings with W-L records and win percentages
  Live Scores & Schedule: Today's games with live/final scores and date filtering
  Player Comparison: Side-by-side stat comparison with animated visual bars
  2K-Inspired UI: Card-based layouts, dark theme with bold accent colors, smooth animations

Tech Stack
  Next.js
  React
  Tailwind CSS
  JavaScript

APIs
ESPN NBA API (Primary data source)
  Scoreboard: http://site.api.espn.com/apis/site/v2/sports/basketball/nba/scoreboard
  News: http://site.api.espn.com/apis/site/v2/sports/basketball/nba/news
  All Teams: http://site.api.espn.com/apis/site/v2/sports/basketball/nba/teams
  Specific Team: http://site.api.espn.com/apis/site/v2/sports/basketball/nba/teams/:team

Pages
  Home: Hero section with spotlight player, today's games, latest news, quick navigation
  Players: Search with autocomplete, filter options, grid of player cards

  Player Detail: Hero card with player image, season statistics, animated stat bars, career highlights
  Standings: Conference toggle tabs, ranked table with color-coded rows
  Scores: Date selector, game cards with live scores, auto-refresh for live games
